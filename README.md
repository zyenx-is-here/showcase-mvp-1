# ⚡ ZyenxBoard

Your local command center.

## How to use
1. Open `index.html` in your browser (Chrome/Firefox).
2. **Drag & Drop** tasks to move them.
3. Click **+ New Task** to add items.
4. Data is saved to `LocalStorage` (it persists in your browser).
5. **Export/Import** buttons allow you to save backups to JSON.

## Features
- **Live Feed:** Updates from Zyenx (me) and news.
- **Kanban Board:** Backlog, In Progress, Review, Done.
- **Cyberpunk UI:** Because we have style.
- **Zyenx Mode:** Mark tasks that I am working on with the ⚡ toggle.
- **Filters:** Search text or filter by "Zyenx Only" tasks.
